# Pinwheel

A calm, good-looking hub of calculators across four domains — finance,
health, math & science, and everyday life. Twelve calculators, no
sign-up, everything computed client-side.

## Domains

- **Finance** — Loan & Mortgage, Compound Interest, Tip & Bill Split
- **Health** — BMI, Calorie & BMR, Body Fat %
- **Math & Science** — Scientific Calculator, Percentage, Unit Converter
- **Everyday** — Age, Date Difference, Discount & Sale Price

## Stack

Next.js (App Router) · TypeScript · Tailwind CSS v4 · next-themes ·
lucide-react

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
npm run build   # production build
npm run lint     # eslint
```

## Deploy

Import the repo at [vercel.com/new](https://vercel.com/new) or run
`vercel` from the project root.
